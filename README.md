# **APPLYZER** - **APK ANALYSIS TOOL**

APPLYZER is an open-source malware analysis tool for detection malicious android packages (APK).

# **Prerequisite**

    1.LINUX 

    2.Androguard

    3.Python

# **USAGE**
    1.git clone repo
    2.cd APPLYZER
    3.pip install -r requirements.txt
    4.python Applyzer.py
